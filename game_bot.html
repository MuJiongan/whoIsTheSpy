Commands:



$play:

Start the game

$add @player:

Add players to the game

$getlist:

Get the list of players who are currently playing


$start:

After adding all the players, start the first round of the game

$changeword:

change the current word if you got the repeated word

$vote @player:

After everyone finishes speaking, vote for someone who you think is the spy


$getvote:

Get the current vote

$clearvote:

Clear all the votes(if someone makes a mistake)

$kick:

kick out the player who has the most votes
